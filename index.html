VA Resources

This is a live, working repository to prove machine readable access to Department of Veteran Affairs resources. Currently this is not an official VA project, but seeks to get official approval of the federal agency.

The data represented below is available as a full JSON, CSV and XML downloads or you can access via a web API, which allows you to query against the data without donwloading and parsing into your own system.

This is a beta project, on how to work with government data, make it available to the public, and at this point offers no service level agreement (SLA) for availability or quality. The goal is to make it as accessible as possible and open to contribution by the public.

This entire project is hosted as a Github repository, use the links above to fork entire project or you can access the data sources directly in the master repository.
